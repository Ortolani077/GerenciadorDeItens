# GerenciadorDeItens

Projeto dividido em duas branches principais para organização entre backend e frontend.

## Branches do Projeto

### Backend

Branch principal contendo a API Java com Spring Boot:

```bash
main
```

Tecnologias utilizadas:

* Java 21
* Spring Boot
* Spring Data JPA
* H2 Database
* Maven/Gradle

---

### Frontend

Branch contendo a aplicação Angular:

```bash
frontend
```

Tecnologias utilizadas:

* Angular
* TypeScript
* HTML/CSS

---

## Como acessar cada parte

### Clonar o projeto

```bash
git clone https://github.com/Ortolani077/GerenciadorDeItens.git
```

---

### Acessar Backend

```bash
git checkout main
```

---

### Acessar Frontend

```bash
git checkout frontend
```
